# How To Start

1.	<b>Node Environment</b></br>
	curl -sL https://deb.nodesource.com/setup_5.x | sudo bash -</br>
	sudo apt-get install -y nodejs

2.	<b>Install Module</b>
	*	npm install bleno
	*	npm install --save datauri
	* 	npm install raspicam
	* 	npm install rpi-gpio

3.	<b>Run</b></br>
	sudo node main.js
